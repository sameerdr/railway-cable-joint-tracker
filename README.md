# Railway Cable Joint Tracker

Android app for tracking railway cable joint locations with Google Sheets integration.

## Features

- **Add Data Form**: Record cable joint information with auto geo-location
- **Show Joint Info**: View and edit all previous entries
- **Show Map**: Display joint locations on Google Maps
- **Reports**: Generate sorted reports and export to Excel, Word & PDF

## Setup Instructions

1. Clone this repository
2. Open in Android Studio
3. Update `google-services.json` with your Firebase config
4. Update Google Sheets API credentials
5. Replace `GOOGLE_SHEET_ID` in `Constants.java` with your sheet ID: `1hjViC-yDssz7KPelBYr-2AEDQXm7BU08VG1GyFkB--w`
6. Build and run

Your data sheet: https://docs.google.com/spreadsheets/d/1hjViC-yDssz7KPelBYr-2AEDQXm7BU08VG1GyFkB--w/edit
